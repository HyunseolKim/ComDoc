ComDoc이란?

ComDoc이란 'Computer Doctor'의 준말로 컴퓨터 출장수리 서비스를 앱서비스와 연계하여, 서비스를 평가하고 견적을 비교하여 결정할 수 있는 양방향 플랫폼 입니다. 신뢰할 수 없으며 비싸고 천차만별인 컴퓨터 수리비, ComDoc에서 먼저 비교하고 지불하세요.

1) 스마트폰 컴닥 앱 또는 컴닥 홈페이지에서 컴퓨터 고장이슈를 올리고 원하는 지역을 선택한다.

2) 선택한 지역에 있는 컴닥 파트너 업체들로부터 빠른 시간 안에 수리사례별 견적이 도착한다.

3) 마음에 드는 업체를 직접 선택해서 수리를 받는다.
